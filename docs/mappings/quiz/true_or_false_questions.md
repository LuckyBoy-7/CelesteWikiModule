# 判断题

题目来源: 底龙

---

1. Everest 是 Celeste 的地图 Mod 编辑器 (错误)
2. Celeste 的本体地图支持用 Loenn 打开 (正确)
3. Celeste 地图 Mod 需要在文件夹内放置完整的 `everest.yaml` 作为使用 Loenn 打开地图编辑的前置条件 (错误)
4. 地图只能使用 File 工具栏中的 Save 或者 Save As 进行保存 (错误)
5. Loenn 内支持使用 `Ctrl + Z` 对误放置的实体进行撤销 (正确)
6. Loenn 当前的最新版本为 1.0.5 (正确)
7. Loenn 的 1.0.5 版本不支持直接拖拽房间进行房间移动 (错误)
8. 所有的实体本身都可以进行材质更换 (错误)
9. 在地图中更改物体的属性时必须选中物体后, 右键打开其属性窗口 (错误)
10. 因为一些原因导致 Loenn 内不能正常显示的实体不能对其进行修改 (错误)
11. 根据常用形状, Loenn 将绘制地图地形砖的工具分为笔刷、油漆桶、圆形、椭圆形、直线、以及矩形 (正确)
12. Selection 是选择工具, 一般对于实体或者 Trigger 都建议使用 Selection 进行精确选择操作 (正确)
13. Loenn 本体不支持插件(scripts)读取 (正确)
14. 地图元数据信息, 比如音乐、背景音等设置, 应该在 Loenn 的工具栏中找到 View -- Metadata, 然后对相应项目进行修改 (错误)
15. 进入地图时, 如果不经过特别设置, 一般情况玩家的入场动画由 Custom Wipe 决定 (错误)
16. 想要地图初始最大冲刺数为 2, 使用 Metadata 是不能实现的 (错误)
17. 在章节选择界面, 地图的名字文本颜色可以更改 Metadata 中的 Title Text Colour (正确)
18. 如果我想要对放置于地图中的原版绿泡泡进行材质更换, 最好的做法是将更换的材质的贴图放到和原版结构相同的路径下 (错误)
19. 只要使用 Loenn, 我就一定能补全我的地图缺失的所有依赖 (错误)
20. Celeste 地图 Mod 的音乐文件使用 FMOD 整合于 Mod 之中 (正确)
21. 在地图内自定义音乐的步骤是将音乐的 `.mp3` 文件放置于 Audio 文件夹, 之后进入 Loenn 对音乐文件进行选择, 保存后即可 (错误)
22. 在地图中吃掉水晶之心仍然不能结束关卡是因为 Metadata 选项设置有误而不是水晶之心本身特性 (正确)
23. 在新建房间时想要保留和上一个房间一样的属性设置, 只需要选中上一个房间, 然后使用 Rooms 工具新建房间即可 (正确)
24. Loenn 的本体支持复制房间 (错误)
